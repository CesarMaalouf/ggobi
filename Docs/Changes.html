This file gives a list of the changes for each of the "versions" of GGobi. It is organized in reverse order so that the most recent releases are We hope to keep it as up-to-date as possible, but we often forget.

Version 0.9.4

  • Added a row identifier argument to setData() for supporting edges.
  • Command line arguments that take a value can be specified as 
           --argName=value
    as well as 
           -argName value
  • -noinit command line argument for ignoring initialization file.
